Basic Variants

Primary Button

Main actions with green glow effect

<button class="btn btn-primary"> <i class="fas fa-check"></i> Save Changes </button>

Secondary Button

Alternative actions with blue glow

<button class="btn btn-secondary"> <i class="fas fa-times"></i> Cancel </button>

Danger Button

Destructive actions with red glow

<button class="btn btn-danger"> <i class="fas fa-trash"></i> Delete </button>

Ghost Button

Minimal style, slides right on hover

<button class="btn btn-ghost"> <i class="fas fa-arrow-right"></i> Learn More </button>

Usage Notes

How to use these buttons in your project:

  • 1. Make sure you include the design system CSS files in your HTML
  • 2. Copy the HTML code from any button above
  • 3. All styles are already defined in static/css/buttons.css
  • 4. Mix and match variants and sizes as needed
  • 5. Add Font Awesome icons for better visual feedback
